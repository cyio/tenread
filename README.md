# 十阅 TenRead 

> 骐骥一跃,不能十步;驽马十驾,功在不舍。

随时获取网站的热门文章10篇，充分利用碎片化时间。

## 贡献自定义内容源

Fork 此项目，修改 data 目录下的 *.json 并 pull request。

## 安装
`git clone https://github.com/cyio/tenread.git`
进入`src`文件夹，运行`bower install`，安装依赖包。然后在Chrome中加载即可。
